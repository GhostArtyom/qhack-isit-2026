ISIT26 Quantum Hackathon – Project Ideas

🔹 Quantum-Enhanced Channel Coding & Error Correction
Objective: Design quantum-inspired techniques for robust channel coding and error correction in classical communication systems.
Expected Outcomes:
Propose and test new quantum-inspired codes or decoding strategies
Demonstrate improvements in error resilience or decoding complexity
Prototype implementations using quantum simulators or hybrid models

🔹 Quantum Machine Learning for Signal Processing
Objective: Apply quantum machine learning (QML) to complex signal processing tasks such as noise reduction, signal detection, or pattern recognition.
Expected Outcomes:
Use QML algorithms (e.g., quantum classifiers or autoencoders) for real-world signal datasets.
Compare performance with classical baselines
Showcase the adaptability of quantum techniques to noisy or high-dimensional data

🔹 Quantum Optimization for Network Resource Allocation
Objective: Employ quantum optimization methods (e.g., QAOA, quantum annealing) to solve network-level challenges like spectrum allocation and traffic routing.
Expected Outcomes:
Design optimization models mapping network tasks to quantum formalisms (e.g., QUBO)
Explore computational gains in large or dynamic network settings
Demonstrate hybrid simulation results with interpretable outcomes

🔹 Quantum Secure Authentication Protocols with Privacy Protection
Objective: Develop advanced authentication protocols using quantum security primitives like QKD, quantum hashes, or post-quantum signatures.
Expected Outcomes:

Create protocols for use in applications like e-commerce and secure communication
Simulate attacks and defenses, and analyze privacy guarantees
Assess performance, integration with classical systems, and scalability

🔹 Hybrid Quantum-Classical Maximum-Likelihood Detection
Objective: Implement Grover-based adaptive search for efficient signal detection in reconfigurable intelligent surface (RIS)-assisted wireless systems.
Expected Outcomes:
Apply Grover’s algorithm to structured search in communication models
Develop hybrid detection schemes blending quantum and classical inference
Benchmark against existing methods and demonstrate detection improvements

🔹 Quantum Machine Learning for Edge Detection in Images
Objective: Explore the use of QML models for classical edge detection tasks in image processing, such as identifying object boundaries in low-resolution or noisy data.
Expected Outcomes:
Design a QNN or quantum convolution model for binary edge map generation
Apply to benchmark datasets (e.g., MNIST, medical scans, satellite imagery)
Compare with classical edge detectors (Sobel, Canny) in accuracy and noise robustness
Explore encoding schemes (angle, amplitude, qubit-efficient) for image patches

🔹 Quantum Models for Corner Detection & Keypoint Extraction
Objective: Implement QML techniques to identify salient keypoints (e.g., corners, junctions) in images, which are useful for visual SLAM, AR/VR, and robotics applications.
Expected Outcomes:
Propose a quantum circuit model that maps image gradients or Hessian-like structures into a QNN
Benchmark performance on corner detection datasets (e.g., using Harris or FAST as baselines)
Investigate how entanglement and superposition improve local-global feature encoding
Highlight applicability to resource-constrained devices (e.g., quantum edge AI)

✅ Participants are encouraged to explore these or propose original project ideas aligned with quantum computing, learning, optimization, or communication theory.
